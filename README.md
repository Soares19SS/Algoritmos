💻 Exercícios de Programação em C
Este repositório foi criado para armazenar e organizar todos os meus exercícios, desafios e projetos desenvolvidos enquanto aprendo a linguagem C. O objetivo é consolidar conceitos fundamentais, desde lógica de programação até estruturas de dados mais complexas.

🛠️ Tecnologias e Ferramentas
Linguagem: C

Compilador: GCC (GNU Compiler Collection)

Ambiente de Desenvolvimento: VS Code / Terminal

🚀 Como Executar os Exercícios
Para compilar e rodar qualquer programa deste repositório, você precisará de um compilador C instalado.

Clone o repositório:

Bash
git clone https://github.com/seu-usuario/nome-do-repositorio.git
Acesse a pasta do exercício:

Bash
cd nome-do-repositorio/pasta-do-exercicio
Compile o código:

Bash
gcc nome-do-arquivo.c -o programa
Execute:

Bash
./programa
